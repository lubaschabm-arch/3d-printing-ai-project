# ИИ-проект: Контент для 3D-печати в ВКонтакте

## Описание
Автоматизация создания контента для продвижения услуг 3D-печати с использованием ИИ-инструментов.

## Инструменты
- **Qwen (LLM)** — генерация текстов
- **Kandinsky 3.0** — создание визуалов

## Цель
Сократить время на создание контента для ВК и повысить конверсию в заявки.

## Структура
- `01_prompts/` — промпты с комментариями
- `02_screenshots/` — фиксация процесса
- `03_results/` — результаты генерации
- `04_final_report/` — итоговый отчёт
